# Bybit Spot Grid Screener

Программный комплекс на Python для автоматического анализа рынка **Bybit Spot** и поиска наиболее эффективных активов для запуска сеточных (Grid) ботов. Программа анализирует текущую рыночную фазу, волатильность и ликвидность, предоставляя готовые параметры для настройки бота.

## 1. Функциональные возможности
Программа выполняет следующие действия:
* **Скан рынка:** Получает список всех активных спотовых пар к USDT.
* **Анализ тренда:** Рассчитывает индекс ADX (14) для определения отсутствия направленного движения (флета).
* **Анализ волатильности:** Рассчитывает нормализованный %ATR для оценки "размаха" микро-колебаний.
* **Фильтрация по ликвидности:** Отсеивает активы с 24-часовым объемом торгов ниже заданного порога.
* **Оптимизация параметров сетки:** Автоматически подбирает количество сеток и границы диапазона под конкретный бюджет и целевую прибыль.

## 2. Алгоритм и математические формулы

### А. Ценовой диапазон (Range)
Диапазон строится на основе средней волатильности актива за последние 300 свечей. Используется множитель $\times 3$ для покрытия наиболее вероятных отклонений цены.

1.  **Ширина диапазона (%)**:
    $$\text{range\_width\_pct} = \%ATR \times 3$$
2.  **Нижняя цена**:
    $$P_{lower} = P_{close} \times (1 - \frac{\text{range\_width\_pct}}{200})$$
3.  **Верхняя цена**:
    $$P_{upper} = P_{close} \times (1 + \frac{\text{range\_width\_pct}}{200})$$

### Б. Расчет количества сеток (Grids)
Программа стремится максимизировать количество сделок при сохранении минимально допустимой прибыли с каждого круга.

1.  **Целевая грязная прибыль**:
    $$\text{Target Gross} = \text{MIN\_NET\_PROFIT} + \text{EXCHANGE\_FEE}$$
2.  **Теоретическое количество сеток**:
    $$\text{Calculated Grids} = \frac{\text{range\_width\_pct}}{\text{Target Gross}}$$
3.  **Ограничение по бюджету**:
    Чтобы избежать слишком мелких ордеров, количество сеток ограничивается бюджетом (минимум 2 USDT на уровень):
    $$\text{Max Grids} = \frac{BUDGET}{2}$$
    $$\text{Final Grids} = \min(\text{Calculated Grids}, \text{Max Grids})$$

### В. Итоговая прибыль на сетку
После расчета финального количества уровней вычисляется реальная чистая прибыль (уже за вычетом комиссии):
$$\text{Net Profit \%} = \left( \frac{\text{range\_width\_pct}}{\text{Final Grids}} \right) - \text{EXCHANGE\_FEE}$$

## 3. Критерии фильтрации (Конфигурация)

Для того чтобы актив попал в список подходящих, он должен соответствовать следующим условиям:

| Параметр | Условие | Описание |
| :--- | :--- | :--- |
| **ADX** | $< 25$ | Рынок находится в стадии бокового движения (флета). |
| **%ATR** | $\ge 0.3\%$ | Достаточная волатильность для перекрытия комиссий. |
| **Volume 24h** | $\ge 5,000,000$ | Высокая ликвидность для быстрого исполнения ордеров. |
| **Final Grids** | $\ge 10$ | Плотность сетки позволяет вести активную торговлю. |

## 4. Технические детали
* **Таймфрейм:** 15 минут (оптимально для среднесрочного Grid-бота).
* **Объем данных:** 300 последних свечей для стабильного расчета индикаторов.
* **Метод сглаживания:** Wilder's Smoothing (через EWM) для ADX.
* **API:** Используется библиотека `ccxt` для работы с биржей Bybit.

## 5. Запуск
1. Установите зависимости: `pip install ccxt pandas numpy`
2. Запустите скрипт: `python main.py`
3. Введите `y` в консоли для начала сканирования.
